How this works: you do the simple steps below (upload clips, build the calendar), then send Gio the links. He wires them into the site once, and after that it stays current on its own. The site link never changes: rfinlay.github.io/camron-finlay-baseball.

🎥 Part 1 — Highlight Videos (YouTube)

Goal: a clean library of Camron's best clips that plays right on his website.

  1. Go to youtube.com and sign in with a Google account. If Camron does not have one yet, create a free account first — this becomes his channel.
  2. Create the channel (first time only): click your profile picture in the top right, choose Create a channel, and name it Camron Finlay Baseball.
  3. Click the Create button (the video-camera icon, top right), then Upload video.
  4. Drag in a clip and give it a clear title, for example "Camron Finlay — Home Run vs. Brighton Bulldogs 12U".
  5. On the visibility step, choose Unlisted. This means anyone with the link can watch, but it will not show up in public search. Click Publish.
  6. When it finishes, click Share and Copy the video link.
  7. Repeat for each highlight. Keep the best 3 to 6 clips — for recruiters, quality beats quantity.
  8. Send the links to Gio. He embeds them in the Highlights section of the site.
Pro tips: Put the clips in a Playlist called "Camron Highlights" to keep them organized as one library. Lead with the home runs and hardest-hit plays. A clear title with the opponent and year helps coaches.

📅 Part 2 — Game Calendar (Outlook)

Goal: games you add in Outlook appear on the website automatically, forever.

  1. Go to outlook.com/calendar and sign in.
  2. Create a calendar: in the left panel click Add calendar → Create blank calendar, name it Camron Baseball, and save.
  3. Add the games as events on that calendar. Put the opponent in the title, and set the date, time, and location (field) for each game.
  4. Publish the calendar: click the Settings gear (top right) → View all Outlook settings → Calendar → Shared calendars. Under "Publish a calendar," pick Camron Baseball, set permission to Can view all details, and click Publish.
  5. Two links appear. Copy the one ending in .ics (the ICS link).
  6. Send the ICS link to Gio. He connects it to the site one time.
  7. That is it. From now on, any game you add to the "Camron Baseball" calendar in Outlook shows up on the website and in the Upcoming Games list automatically. No more updates needed.
Pro tip: Keep using the same "Camron Baseball" calendar for every season. Add, move, or cancel games in Outlook and the site follows along on its own.
